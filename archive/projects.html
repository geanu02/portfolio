Photo Store Web Application

Bird's Eye View: My Photo Collection & Store

Built the frontend prototype of a photo gallery utilizing React, StyledComponents and Heroku functions that showcase photos, enabling favorites and carts, and online ordering. Deployed custom API using Heroku functions to share my personal collection of stunning photos from hikes and memorable walks. Applied Responsive Design practices, React custom hooks and router, and deployed the web application on Netlify through Github to enable CI/CD.

Movie Search Web Application

A Cinematic Titles Engine

Built the frontend UI of a movie database search engine using React that enables searching for titles with synopsis and release date information. Under the hood, the API from The Movie DB was utilized to extract movie data titles and information. Applied modern web standards in Single-Page Application (SPA) using React.js functional components, built-in/custom hooks and JavaScript ES6+ for fetch calls. Utilized the MaterialUI design system for forms and Jest for testing functions. Deployed the web application on Netlify through Github to enable CI/CD.
